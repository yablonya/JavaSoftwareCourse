In method `"main"` set your matrices in variables `"matrix1"` and `"matrix2"`\
```java

                    {1, 40, 2},
                    {16, 12, 6},
                    {7, 5, 9}
                   };

byte[][] matrix2 = {
                    {9, 8, 7},
                    {6, 5, 4},
                    {3, 2, 1}
                   };
```
and then run your code
